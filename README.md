# MYKORA — Система роста

Современная воронка продаж с минималистичным дизайном.

## Возможности

- 🎨 Современный и минималистичный дизайн
- 📱 Полностью адаптивный (мобильный, планшет, десктоп)
- ⚡ Быстрая загрузка (одного HTML файл)
- 🎯 Оптимизирован для конверсий

## Развёртывание

### На Vercel (рекомендуется)

1. Отправьте репозиторий на GitHub
2. Подключите Vercel к вашему GitHub аккаунту
3. Импортируйте репозиторий в Vercel
4. Готово! Ваш сайт будет развёрнут

### Локально

```bash
# Python 3
python -m http.server 8000

# Затем откройте http://localhost:8000
```
